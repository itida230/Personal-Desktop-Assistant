Tom - Your Personal Desktop Virtual Assistant:

A simple Python-based voice assistant that can recognize speech, respond with text-to-speech, search Wikipedia, play music, and open popular websites.

Key Features

1) Voice Recognition
Uses speech_recognition to capture user voice commands.
Converts speech to text using Google Speech Recognition API.

2) Text-to-Speech (TTS)
Uses pyttsx3 for speech output.
Greets the user based on the current time of day.

3) Personalization
Customizable user name.
Handles queries like "What is my name?" and responds personally.

4) Wikipedia Search
Searches Wikipedia for the requested topic.
Reads out a two-sentence summary.
Handles missing or ambiguous results gracefully.

5) Time Query
Tells the current system time when asked.

6) Website Shortcuts
Opens common websites in the browser such as YouTube, Google, Netflix, and Spotify.

7) Music Playback
Plays music from a predefined local folder.
Automatically plays the first available song.

8) Conversation Handling
Recognizes simple greetings like "hello", "hi", "good morning".
Responds to queries such as "who are you" and "how are you".

9) Exit Command
Terminates the program on commands like "exit", "quit", or "stop".

10) Error Handling
Handles cases where voice is not recognized.
Provides safe handling of Wikipedia errors and missing files.
